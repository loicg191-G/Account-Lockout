<h1>Account Lockout</h1>
This tutorial shows how to unlock a user's account that has been locked due to too many failed login attempts.
<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Active Directory

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>Unlocking a Locked Account</h2>
<p>
  <h2>Step 1: Finding the Account</h2>

 When the "Find Users, Contacts and Groups" window opens, fill the "Name" field ("fox.civ" in this instance) and click "Find Now".
</p>
   <h2>Step 2: Unlocking the Account</h2>

From there, click "Account", check the "Unlock account..." box then click Apply. Confirm by clicking "OK" and the account should be unlocked and ready to be logged back in again.
